# Skill：回覆陌生開發

## 用途

處理外部陌生人或公司主動聯繫的業務開發、服務推銷、合作推薦等郵件。

## 流程

1. 從來信中提取：
   - 寄件人/公司名稱
   - 推銷什麼服務/產品
   - 合作提案內容
   - 是否為群發郵件（判斷個人化程度）
4. 判斷回覆方向：
   - 是否與團隊需求相關
   - 是否值得進一步了解
   - 是否為明顯的垃圾郵件/群發

## 回覆方向

- **婉拒** → 使用 `模板庫/陌生開發/婉拒-標準.md`（禮貌拒絕）
- **有興趣** → 使用 `模板庫/陌生開發/有興趣-了解更多.md`（請提供更多資訊）
- **不回覆** → 如果是明顯的垃圾郵件，建議用戶不需回覆

## 輸出格式

```
回信草稿（陌生開發）

來源：{公司/個人名稱}
推銷內容：{簡述}
判斷：{值得回覆/禮貌婉拒/建議不回覆}

主旨：{回信主旨}

---

{回信正文}

---

請審核以上內容，確認後可直接發送。
```
