# Skill：回覆觀眾來信

## 用途

處理觀眾純粹分享感想、感謝鼓勵、或來罵人的郵件。

## 流程

1. 判斷子類型：
   - **分享感想** → 回覆感謝分享
   - **感謝鼓勵** → 回覆感謝支持
   - **罵人吵架** → 通常不需回覆，告知用戶

## 子類型處理

### 分享感想
- 使用 `模板庫/觀眾來信/感謝分享.md`
- 感謝對方特別來信，表示會轉達團隊

### 感謝鼓勵
- 使用 `模板庫/觀眾來信/感謝鼓勵.md`
- 表達真誠的感謝，這對團隊是很大的鼓勵

### 罵人吵架
- 通常建議不需回覆
- 如果用戶堅持要回，保持專業冷靜

## 輸出格式

直接輸出可複製的回信內容，不加任何框線或標記。
