# Skill：回覆觀眾勘誤

## 用途

處理觀眾指出影片或內容中事實錯誤、資料問題的郵件。

## 流程

1. 從來信中提取：
   - 觀眾名稱
   - 指出哪個影片/內容有錯
   - 錯誤的具體內容是什麼
   - 觀眾提供的正確資訊/來源
4. 判斷回覆方向：
   - 錯誤是否屬實（需用戶確認）
   - 是否需要在影片中加註修正
   - 是否為觀點差異而非事實錯誤
5. 根據判斷選擇模板，產生回信草稿

## 回覆方向

- **確認修正** → 使用 `模板庫/觀眾勘誤/感謝-確認修正.md`（感謝指出、說明會修正）
- **說明原因** → 使用 `模板庫/觀眾勘誤/感謝-說明原因.md`（感謝關注、解釋為什麼這樣呈現）

## 輸出格式

```
回信草稿（觀眾勘誤）

勘誤摘要：{觀眾指出什麼問題}
影響範圍：{哪個影片/內容}
回覆方向：{確認修正/說明原因}

主旨：{回信主旨}

---

{回信正文}

---

請審核以上內容，確認後可直接發送。
需要在影片中加註修正嗎？
```
