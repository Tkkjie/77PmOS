# Skill：回覆 YouTube 官方

## 用途

處理來自 YouTube 團隊的官方通知、邀請、政策更新、合作項目等郵件。

## 流程

1. 從來信中提取：
   - 寄件人（YouTube 哪個部門/團隊）
   - 郵件類型（通知/邀請/政策更新/合作項目/其他）
   - 需要回覆或行動的具體內容
   - 時間要求/截止日期
4. 判斷回覆方向：
   - 是否需要回覆（有些通知不需要）
   - 回覆的優先級
   - 是否需要團隊其他人協助處理

## 回覆方向

- **標準回覆** → 使用 `模板庫/YouTube官方/標準回覆.md`
- **不需回覆** → 告知用戶這是通知型郵件，不需要回覆
- **需要討論** → 摘要重點，建議與團隊討論後再回覆

## 輸出格式

```
回信草稿（YouTube 官方）

來源：{YouTube 部門/團隊}
類型：{通知/邀請/政策更新/合作項目}
判斷：{需回覆/不需回覆/需討論}

主旨：{回信主旨}

---

{回信正文}

---

請審核以上內容，確認後可直接發送。
```
